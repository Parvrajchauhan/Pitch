## ЁЯУМ What Is This?

**Pitch Visualizer** is a full-stack AI service that transforms a block of narrative text тАФ a customer success story, a startup pitch, a product description тАФ into a **coherent, multi-panel visual storyboard**.

It does three things well:
1. **Segments** your text into logical scenes using NLP sentence tokenization (NLTK)
2. **Engineers** a rich, visually descriptive prompt for each scene using an LLM (Gemini)
3. **Generates** a unique image per panel via a text-to-image API and presents everything as a clean, sequential storyboard UI

 It is a complete **idea тЖТ narrative тЖТ visuals** pipeline.

## тЬи Features
- ЁЯза **NLP-based sentence segmentation** тАФ splits narrative into meaningful, standalone scenes
- ЁЯОи **LLM-powered prompt refinement** тАФ Gemini transforms each sentence into a richly descriptive visual prompt, adding cinematic framing, lighting, and mood
- ЁЯЦ╝я╕П **AI image generation** тАФ calls a text-to-image API (Stable Diffusion / DALL┬╖E) for each engineered prompt
- ЁЯОЮя╕П **Multi-panel storyboard UI** тАФ panels are rendered sequentially with original text captions
- ЁЯОн **User-selectable visual styles** тАФ choose from `cinematic`, `anime`, `Fantasy`, `realistic`, or `Sketch` before generating
- ЁЯР│ **Fully containerized** тАФ Docker Compose brings up backend + frontend with a single command

## тЪЩя╕П Setup & Installation

### Prerequisites
- Docker & Docker Compose, **or** Python 3.10+ and Node.js 18+
- A Gemini API key ([get one free](https://aistudio.google.com/app/apikey))
- A HuggingFace API key for image generation

### 1. Clone the Repository

```bash
git clone https://github.com/Parvrajchauhan/Pitch.git
cd Pitch
```

### 2. Configure Environment Variables

```bash
cp server/.env.example server/.env
```

Edit `server/.env`:

```env
GEMINI_API_KEY=
GEMINI_MODEL=gemini-3-flash-preview
HF_API_KEY=
```

### 3A. Run with Docker (Recommended)

```bash
docker-compose up --build
```

- **Backend:** http://localhost:8000
- **Frontend:** http://localhost:3000

## ЁЯза Design & Architecture

### Pipeline

```
Raw Text Input
     тФВ
     тЦ╝
[1] Gemini LLM тАФ Visual Thread Generator
     тФВ  Extracts consistent elements (characters, setting, tone, color palette)
     тЦ╝
[2] NLTK Sentence Tokenizer
     тФВ  Splits into тЙе3 scene segments
     тЦ╝
[3] Gemini LLM тАФ Prompt Engineer
     тФВ  Adds visual framing, lighting, mood, style keywords (using visual thread)
     тЦ╝
[4] Text-to-Image API (Stability AI / DALL┬╖E)
     тФВ  Generates one image per engineered prompt
     тЦ╝
[5] Storyboard Renderer (React)
     тФВ  Sequential panels, captions, style badge
     тЦ╝
Visual Storyboard Output
```

### Prompt Engineering Strategy

Each sentence is sent to Gemini with a system instruction to return a single, highly descriptive visual prompt. The instruction enforces:
- **Scene framing:** wide shot, close-up, aerial, etc.
- **Lighting descriptors:** golden hour, dramatic shadows, neon, etc.
- **Mood/atmosphere:** tense, hopeful, ethereal, etc.
- **Style keywords** appended from the user's selected preset (e.g., `"cinematic, 35mm film, high detail"`)
