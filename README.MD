## рҹ“Ң What Is This?

**Pitch Visualizer** is a full-stack AI service that transforms a block of narrative text вҖ” a customer success story, a startup pitch, a product description вҖ” into a **coherent, multi-panel visual storyboard**.

It does three things well:
1. **Segments** your text into logical scenes using NLP sentence tokenization (NLTK)
2. **Engineers** a rich, visually descriptive prompt for each scene using an LLM (Gemini)
3. **Generates** a unique image per panel via a text-to-image API and presents everything as a clean, sequential storyboard UI

 It is a complete **idea вҶ’ narrative вҶ’ visuals** pipeline.

## вңЁ Features
- рҹ§  **NLP-based sentence segmentation** вҖ” splits narrative into meaningful, standalone scenes
- рҹҺЁ **LLM-powered prompt refinement** вҖ” Gemini transforms each sentence into a richly descriptive visual prompt, adding cinematic framing, lighting, and mood
- рҹ–јпёҸ **AI image generation** вҖ” calls a text-to-image API (Stable Diffusion / DALLВ·E) for each engineered prompt
- рҹҺһпёҸ **Multi-panel storyboard UI** вҖ” panels are rendered sequentially with original text captions
- рҹҺӯ **User-selectable visual styles** вҖ” choose from `cinematic`, `anime`, `Fantasy`, `realistic`, or `Sketch` before generating
- рҹҗі **Fully containerized** вҖ” Docker Compose brings up backend + frontend with a single command

- Docker & Docker Compose, **or** Python 3.10+ and Node.js 18+
- A Gemini API key ([get one free](https://aistudio.google.com/app/apikey))
- A HuggingFace API key for image generation

### 1. Clone the Repository

```bash
git clone https://github.com/Parvrajchauhan/Pitch.git
cd Pitch
```

### 2. Configure Environment Variables

```bash
cp server/.env.example server/.env
```

Edit `server/.env`:

```env
GEMINI_API_KEY=
GEMINI_MODEL=gemini-3-flash-preview
HF_API_KEY=
```

### 3A. Run with Docker (Recommended)

```bash
docker-compose up --build
```

- **Backend:** http://localhost:8000

## рҹ§  Design & Architecture

### Pipeline

```
Raw Text Input
     в”Ӯ
     в–ј
[1] Gemini LLM вҖ” Visual Thread Generator
     в”Ӯ  Extracts consistent elements (characters, setting, tone, color palette)
     в–ј
[2] NLTK Sentence Tokenizer
     в”Ӯ  Splits into вүҘ3 scene segments
     в–ј
[3] Gemini LLM вҖ” Prompt Engineer
     в”Ӯ  Adds visual framing, lighting, mood, style keywords (using visual thread)
     в–ј
[4] Text-to-Image API (Stability AI / DALLВ·E)
     в”Ӯ  Generates one image per engineered prompt
     в–ј
[5] Storyboard Renderer (React)
     в”Ӯ  Sequential panels, captions, style badge
     в–ј
Visual Storyboard Output
```

### Prompt Engineering Strategy

Each sentence is sent to Gemini with a system instruction to return a single, highly descriptive visual prompt. The instruction enforces:
- **Scene framing:** wide shot, close-up, aerial, etc.
- **Lighting descriptors:** golden hour, dramatic shadows, neon, etc.
- **Mood/atmosphere:** tense, hopeful, ethereal, etc.
- **Style keywords** appended from the user's selected preset (e.g., `"cinematic, 35mm film, high detail"`)
